# Dian Grobbelaar - Prophet, Musician & MMA Artist

Professional website for Dian Grobbelaar featuring his roles as Prophet of God, Musician, and MMA Artist.

## Features

- ✨ Neon Spirituality design theme
- 🎨 Cyan, Magenta, and Lime neon colors
- 📱 Fully responsive design
- 🎬 Premium animations and effects
- 🔗 All social media links integrated
- 💝 PayPal donation button
- 💬 WhatsApp community integration
- 📧 Email contact functionality

## Quick Start

### Option 1: Use the Built HTML File
1. Download `index.html` from the releases
2. Open it directly in your browser
3. No installation needed!

### Option 2: Development Setup
```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
